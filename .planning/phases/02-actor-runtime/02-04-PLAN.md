| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-11 | Repudiation | actor handler exception path | mitigate | Route handler exceptions into the current reply future so callers observe a typed failure rather than silent drops |
| T-02-12 | Spoofing | dead actor handle sends | mitigate | Validate actor liveness/secret-like identity before enqueue and reject sends after stop |
</threat_model>
