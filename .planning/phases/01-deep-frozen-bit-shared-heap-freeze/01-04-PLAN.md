# Phase 1 / Plan 01-04: Atomic-vs-plain refcount branch on `shared`

**Goal**: Replace the unconditional `atomicInc` / `atomicDec` in
`retainManaged` / `releaseManaged` with a per-variant branch on the managed
header's `shared` bit. Shared values keep atomic RC (Phase 0 correctness);
owned values get plain `.inc` / `.dec`, restoring the owned-side performance
that Phase 0 traded for uniformity.

**Locks user decision D-10, D-11**. Independent of 01-02 and 01-03 â€” this plan
can land in parallel with them on top of 01-01.

**Updated Phase 1 decision:** Instance participates in the same `shared`-bit
branch as the other managed headers because `01-01` added `flags` to
`InstanceObj`. The branch therefore applies to `ArrayObj`, `MapObj`, `Gene`,
`Reference`, `String`, and `InstanceObj`.

## Approach

### Helper

Introduce a single cross-variant helper to avoid duplicating the branch
logic per arm:

```nim
template incRc(hdr: ptr, shared: bool) =
  if shared: atomicInc(hdr.ref_count)
  else:      hdr.ref_count.inc()

template decRc(hdr: ptr, shared: bool): int =
  if shared: atomicDec(hdr.ref_count)
  else:      (hdr.ref_count.dec(); hdr.ref_count)
```

(Exact helper shape is at implementer's discretion â€” what matters is that
each arm uses the same pattern.)

### `retainManaged`

For each managed tag case in `src/gene/types/memory.nim:78-170`:

```nim
of 0xFFF8:  # ARRAY_TAG
  let arr = cast[ptr ArrayObj](raw and PAYLOAD_MASK)
  if arr != nil:
    incRc(arr, (arr.flags and SharedBit) != 0)
```

Same shape for MAP, INSTANCE, GENE, REF, STRING.

### `releaseManaged`

Mirror the branch on decrement. The decrement-to-zero branch that triggers
`destroy_*` is unaffected structurally â€” only the decrement primitive
changes.

### Load-ordering invariant

The `shared` bit is only set by the `(freeze v)` op (Plan 01-02) or by
bootstrap publication (`freeze_bootstrap_publication`, already landed in
Phase 0). Both set the bit **before** publishing the pointer cross-thread,
so a reader observing the pointer at all observes `shared == true`
(publication happens-before dereference). Document this invariant in
`memory.nim` as a module-level comment so future phases don't break it.
