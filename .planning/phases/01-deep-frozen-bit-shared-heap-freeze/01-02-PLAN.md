# Phase 1 / Plan 01-02: `(freeze v)` operation and typed errors

**Goal**: Implement `(freeze v)` as a two-pass validate-then-tag recursive
walker (user decision D-06) over the MVP scope `{array, map, hash_map, gene,
bytes}`. Non-MVP kinds raise a typed `FreezeScopeError`. Tagging happens
in-place on the existing heap per D-08.

## Approach

### Module layout

Create `src/gene/stdlib/freeze.nim` housing the walker and public entry point.
Register the `freeze` symbol as a Gene-callable builtin through the existing
stdlib registration path (mirror the registration pattern used for
`String.append` etc. from P0.4).

### Two-pass semantics (D-06)

```nim
type
  FreezeScopeError* = object of CatchableError
    offending_kind*: ValueKind
    path*: string           # e.g. "/children[2]/.name"

proc validate_for_freeze(v: Value, path: string, visited: var HashSet[uint64])
proc tag_for_freeze(v: Value, visited: var HashSet[uint64])

proc freeze_value*(v: Value): Value =
  ## Public entry. Idempotent (already-frozen is a no-op).
  ## Atomic: on validation failure, no bits are set anywhere.
  if v.deep_frozen: return v
  var visited1 = initHashSet[uint64]()
  validate_for_freeze(v, "/", visited1)
  var visited2 = initHashSet[uint64]()
  tag_for_freeze(v, visited2)
  return v
```

### MVP scope handling

- **Leaves** (tag but don't recurse): `VkBytes`, `VkString`.
- **Containers** (tag and recurse): `VkArray`, `VkMap` (iterate values; keys
  already immutable), `VkHashMap`, `VkGene` (walk `type` slot, props values,
  children).
- **Reject** (raise `FreezeScopeError` in pass 1 with offending kind + path):
  `VkInstance`, `VkClass`, `VkBoundMethod`, `VkFunction`, `VkClosure`,
  `VkNativeFn`, any other heap kind not in the MVP list.
- **Accept but no-op** (immutable by construction): `VkNil`, `VkBool`,
  `VkInt`, `VkFloat`, `VkChar`, `VkSymbol`, `VkKeyword`.

Cycle safety: use `visited` set keyed on `raw and PAYLOAD_MASK` (the heap
pointer). Both passes maintain their own set.

### Stdlib registration

Locate the existing stdlib registration surface used by Phase 0 (see
`src/gene/stdlib/core.nim` for the pattern) and register:

```nim
register_builtin("freeze", 1, proc(args: seq[Value]): Value =
  freeze_value(args[0]))
```
